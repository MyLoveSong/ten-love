# 🚀 快速操作卡片

## ⚡ 5分钟快速上手

### 1️⃣ 启动系统
```bash
cd "创新创业/血糖预测/x血糖"
python quick_start.py
```

### 2️⃣ 选择操作
在菜单中选择：
- **1** = 运行基础模型（快速测试）
- **3** = 启动API服务（完整功能）
- **2** = 训练深度学习模型（高性能）

### 🚀 智能集成服务（强烈推荐）
```bash
python smart_integrated_service.py
```
**智能检查已训练模型，有则加载，无则训练**，避免重复训练！

### 🚀 Windows一键启动
```bash
启动智能服务.bat
```
双击运行，自动启动智能服务！

## 🆘 常见问题

### 问题：依赖安装失败
**解决**: 
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

### 问题：端口被占用
**解决**: 
```bash
netstat -ano | findstr :8000
# 或使用其他端口
```

### 问题：内存不足
**解决**: 使用较小的数据集或减少batch_size

## 🎯 推荐流程

### 方式1：分步操作
1. **首次使用**: `python quick_start.py` → 选择 1
2. **测试API**: `python api_service.py` → 访问 http://localhost:8000/docs
3. **训练模型**: `python gluformer_model.py`
4. **查看结果**: 检查生成的 `.png` 和 `.csv` 文件

### 方式2：智能集成服务（强烈推荐）
1. **一键启动**: `python smart_integrated_service.py` 或双击 `启动智能服务.bat`
2. **智能检查**: 系统自动检查是否有已训练模型
3. **快速加载**: 有模型则几秒加载，无模型则自动训练
4. **使用API**: 访问 http://localhost:8000/docs
5. **查看状态**: 访问 http://localhost:8000/training/status

---

> 💡 **小贴士**: 所有命令都在项目目录下运行，确保先 `cd` 到正确位置！
