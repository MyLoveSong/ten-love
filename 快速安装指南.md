# 🚀 快速安装指南

## ❌ 常见错误：`pip install cv2` 失败

**错误原因**：正确的包名是 `opencv-python`，不是 `cv2`

## ✅ 正确的安装命令

### 方法1：一键修复（推荐）
```bash
python fix_imports.py
```

### 方法2：手动安装
```bash
# 安装OpenCV（解决cv2导入问题）
pip install opencv-python

# 安装其他依赖
pip install transformers torch numpy pandas scikit-learn matplotlib Pillow
```

### 方法3：使用requirements.txt
```bash
pip install -r requirements.txt
```

## 🔍 验证安装

## 🆘 常见问题解决

### Q: 安装速度慢怎么办？
A: 使用国内镜像源：
```bash
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple/
```

### Q: 权限错误怎么办？
A: 使用用户安装：
```bash
pip install --user opencv-python
```

### Q: 版本冲突怎么办？
A: 创建虚拟环境：
```bash
python -m venv venv
venv\Scripts\activate  # Windows
pip install opencv-python
```
